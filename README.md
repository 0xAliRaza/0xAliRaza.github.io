# 0xAliRaza.github.io

**Live Preview: [https://0xali.com](https://0xali.com)**

Hi there! This repository contains a one page introductory HTML/CSS template for developers. This is simple but still a slick template for online presence.

## Features 

* SEO optimized.
* Fully responsive.
* SVG icons.
* `Display: flex` used for layout purposes.
* CSS variables 

## Reusability
It's a small project which I didn't want to make any complex by adding some license. So, you're completely free to use it according to your needs. Cheers!
